## ブランチ一覧と現在作業中のブランチ表示
    git branch

## 開発のため作業用ブランチ作成
    git branch develop

## 開発用ブランチへ移動
    git checkout develop


## ローカルリポジトリへ新ファイル追加、コミット
    git add ToDo.txt  
    git commit -m "[add]ToDoファイル追加"  
    git add README.md *.txt

## プッシュして、ローカルの変更をリモートリポジトリに反映する。
    git push origin develop  
    
## 作業用ブランチをメインへマージする。
メインへ移動  
    git checkout master
開発用ブランチをメインへマージ   
    git merge develop
